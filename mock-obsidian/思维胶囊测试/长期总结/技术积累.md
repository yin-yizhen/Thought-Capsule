# 技术积累



### 2026-06-01
- React `useEffect` 依赖项处理：若依赖项为引用类型，为避免引用地址变动导致的无限循环，应将其定义在组件外部或使用 `useMemo` / `useCallback` 进行包裹。


### 2026-06-02
- 技术方案调研：Tauri 框架由于其基于 Rust 的特性，在打包体积上优于 Electron，可作为未来跨平台桌面应用的首选轻量化选型。


### 2026-06-03
- 深色模式配色优化：在设计深色模式时，避免使用纯黑色，采用偏蓝灰色的 `#1e1e2e` 能提供更有质感的视觉体验。


### 2026-06-04
- RAG 架构优化经验：分块 (chunking) 策略是影响召回率的核心因素。在实际应用中，按照语义分块比按字数分块更具可靠性，能有效提升检索阶段的质量。


### 2026-06-05
- VSCode 快捷键：使用 `Ctrl+D` 可多选相同的词，方便快速批量编辑或重构。
